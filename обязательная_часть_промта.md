|---|---|

> **Ключевое правило:** не пиши бизнес-логику сам! Все сервисы (`user_repo`, `campaign_repo`, `billing_service`, `ai_service`, `analytics_service`) реализует belin. Твоя задача — вызывать готовые методы из handlers и форматировать результат для пользователя. Перед началом каждого шага — проверь чеклист Prerequisites.

📊 TGStat API — как получить доступ
🔗 Официальный сайт: https://tgstat.ru
Вариант 1: Официальный API (платный)
TGStat предоставляет платный API для доступа к каталогу каналов.

📍 Где получить:
Перейдите на https://tgstat.ru
Зарегистрируйтесь
Перейдите в раздел API или Для разработчиков
Оформите подписку
💰 Тарифы (ориентировочно):
Базовый — от ~3000₽/мес
Профессиональный — от ~10000₽/мес
Корпоративный — индивидуально
📝 Что даёт API:
Доступ к каталогу каналов по категориям
Статистика каналов (подписчики, охваты, ER)
Поиск по ключевым словам
История изменений статистики
Вариант 2: Бесплатный парсинг (не рекомендуется)
⚠️ Важно: Парсинг сайта TGStat без API нарушает их условия использования и может привести к блокировке IP.

В текущем коде используется TGStatParser который пытается парсить сайт напрямую, но получает 403 Forbidden.

Вариант 3: Альтернативы TGStat
Если TGStat API слишком дорогой, рассмотрите альтернативы:

Сервис	API	Цена
Telemetr.me	✅ Есть	~2000₽/мес
Tgstat.ru	✅ Есть	~3000₽/мес
TelegramChannels.me	❌ Нет	Бесплатно
Combot.org	✅ Есть	Есть бесплатный тариф

📝 Если решите подключить TGStat API:
Получите API ключ на https://tgstat.ru/api
Добавьте в .env:
TGSTAT_API_KEY=your_tgstat_api_key_here
Обновите docker-compose.yml:
environment:
  TGSTAT_API_KEY: ${TGSTAT_API_KEY}
Перезапустите бота:
docker compose restart bot worker
В коде src/utils/telegram/tgstat_parser.py добавьте использование ключа в заголовки запросов.
💡 Альтернативный подход:
Вместо TGStat можно использовать встроенный поиск Telegram более эффективно:

Увеличить количество поисковых запросов в SEARCH_QUERIES
Использовать разные языки (русский, английский)
Добавить больше конкретных тем (ai, crypto, startups, и т.д.)
Это бесплатно и не требует внешних API!
