## Обзор навигации

### Типы меню

| Тип меню | Когда показывается | Файл клавиатуры |
|----------|-------------------|-----------------|
| **Онбординг** | Новый пользователь без каналов и кампаний | `main_menu.py::get_onboarding_kb()` |
| **Рекламодатель** | Есть кампании, нет каналов | `main_menu.py::get_advertiser_menu_kb()` |
| **Владелец** | Есть каналы, нет кампаний | `main_menu.py::get_owner_menu_kb()` |
| **Комбинированное** | Есть и каналы, и кампании | `main_menu.py::get_combined_menu_kb()` |
| **Админ** | Пользователь в ADMIN_IDS | `admin.py::get_admin_main_kb()` |

### Определение роли

```python
# UserRoleService.get_user_context(user_id)

has_channels = await user_repo.has_channels(user_id)
has_campaigns = await campaign_repo.has_campaigns(user_id)

if has_channels and has_campaigns:
    role = "both"
elif has_channels:
    role = "owner"
elif has_campaigns:
    role = "advertiser"
else:
    role = "new"
```

| Кнопка | Callback Data | Действие |

### FSM создания кампании (вручную)

**Файл:** `src/bot/states/campaign.py::CampaignStates`

**Цепочка состояний:**

```
waiting_title
    ↓
waiting_header (опционально)
    ↓
waiting_text
    ↓
waiting_image (опционально)
    ↓
waiting_topic
    ↓
waiting_member_count
    ↓
waiting_schedule
    ↓
waiting_confirm
    ↓
[Запуск кампании]
```

**Кнопки на каждом шаге:**

| Шаг | Кнопки действий | Кнопки навигации |
|-----|----------------|------------------|
| waiting_title | — | 🔙 В меню |
| waiting_header | ➡️ Пропустить | 🔙 Назад, 🔙 В меню |
| waiting_text | ✍️ Ввести вручную, 🤖 AI | 🔙 Назад, 🔙 В меню |
| waiting_image | 📷 Загрузить, ➡️ Пропустить | 🔙 Назад, 🔙 В меню |
| waiting_topic | Выбор темы (inline) | 🔙 Назад |
| waiting_member_count | Выбор диапазона | 🔙 Назад |
| waiting_schedule | ⏰ Сейчас, 📅 Позже | 🔙 Назад |
| waiting_confirm | ✅ Запустить, 📝 В черновик | 🔙 Назад, ✏️ Редактировать |

### AI создание кампании

**Файл:** `src/bot/handlers/campaign_create_ai.py`

**Цепочка состояний:**

```
waiting_topic
    ↓
waiting_ai_description
    ↓
waiting_variants (AI генерирует 2-3 варианта)
    ↓
waiting_url (опционально)
    ↓
waiting_image
    ↓
waiting_confirm
    ↓
[Запуск кампании]
```

**Особенности:**
- AI генерирует текст на основе темы и описания
- Пользователь выбирает лучший вариант
- Можно запросить новые варианты
- Можно отредактировать выбранный текст

## Навигационные цепочки

### Путь рекламодателя

```
/start → Онбординг → "Хочу размещать рекламу"
    ↓
Меню рекламодателя → "Создать кампанию"
    ↓
Выбор способа → "С помощью AI"
    ↓
AI создание (FSM) → Тема → Описание → Варианты → URL → Фото → Подтверждение
    ↓
[Запуск кампании]
    ↓
Мои кампании → Выбор кампании → Аналитика кампании
```

### Путь владельца канала

```
/start → Онбординг → "У меня есть канал"
    ↓
Меню владельца → "Добавить канал"
    ↓
Добавление канала (FSM) → Username → Цена → Подтверждение
    ↓
Мои каналы → Выбор канала → Заявки → Одобрить/Отклонить
    ↓
[Публикация] → Выплата 80%
```

### Путь администратора

```
/start → Главное меню → "Админ" (если в ADMIN_IDS)
    ↓
Админ панель → "Изменение баланса"
    ↓
Выбор пользователя → Сумма → Причина → Подтверждение
    ↓
[Баланс изменён] → Уведомление пользователю
```
